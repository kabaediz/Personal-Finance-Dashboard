Über das Projekt

Personal Finance Dashboard für Wirtschaftsinformatik-Portfolio

Projektübersicht

Das Personal Finance Dashboard ist eine moderne Webanwendung zur Verwaltung und Analyse persönlicher Finanzen. Es wurde als Portfolio-Projekt für Wirtschaftsinformatik-Studenten entwickelt und demonstriert den Einsatz moderner Webtechnologien ohne große Frameworks.

Die Anwendung ermöglicht es Benutzern, ihre Ausgaben zu erfassen, zu kategorisieren und visuell auszuwerten, um bessere finanzielle Entscheidungen zu treffen.

Tech-Stack

HTML5

Semantische HTML-Struktur für bessere Zugänglichkeit und SEO

Tailwind CSS

Utility-First CSS-Framework für schnelles Styling

Vanilla JavaScript (ES6+)

Moderne JavaScript-Features ohne externe Frameworks

Chart.js

Interaktive Charts für Datenvisualisierung

Features

Kategorisierte Ausgaben

Verwalten Sie Ausgaben in verschiedenen Kategorien wie Lebensmittel, Miete, Freizeit, etc.

Flexible Zeitraum-Filter

Filtern Sie Ihre Daten nach Monat, Quartal, Jahr oder frei wählbaren Zeiträumen

Interaktive Charts

Visualisierung mit Kreisdiagrammen und Balkendiagrammen für bessere Übersicht

JSON-Daten Import

Laden Sie Ausgabendaten aus lokalen JSON-Dateien oder REST-APIs

Einfache Eingabe

Fügen Sie neue Ausgaben direkt über ein intuitives Formular hinzu

Dark Mode

Wechseln Sie zwischen hellem und dunklem Design

CSV Export

Exportieren Sie Ihre Daten als CSV-Datei für weitere Analysen

Responsive Design

Perfekte Darstellung auf Desktop, Tablet und Smartphone

Architektur

JavaScript Module

  • app.js: Haupteinstiegspunkt und Koordinierung aller Module
  • state.js: Zentrale State-Verwaltung für alle Ausgabendaten
  • api.js: Daten-Handling und JSON-Import/Export
  • charts.js: Chart.js Integration und Update-Logik
  • ui.js: DOM-Manipulation und UI-Updates
  • filters.js: Zeitraum-Filterlogik
  • theme.js: Dark Mode und Theme-Management

Schnellstart

Option 1: Direkt im Browser öffnen

Öffnen Sie einfach die index.html Datei in Ihrem Browser.

Option 2: Mit lokalem Webserver

Verwenden Sie einen einfachen Webserver für eine bessere Entwicklungserfahrung:

# Mit Python
python -m http.server 8000

# Mit Node.js (npx)
npx http-server

# Mit VS Code Live Server Extension
Rechtsklick auf index.html → "Open with Live Server"
